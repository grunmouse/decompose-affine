
\subsection{Содержащие $S_x$ или $S_y$}

Матрица $S_x$ или $S_y$ вносит один параметр, следовательно, нужно добавить ещё три матрицы.
Однотипные матрицы не могут идти подряд

\begin{enumerate}
	\item $X$ и $Y$, одна из которых входит дважды
	\begin{enumerate}
		\item $S$ скраю
		\begin{itemize}
			\item $S_x X_1YX_2$
			\item $S_x Y_1XY_2$
			\item $X_1YX_2 S_x$
			\item $Y_1XY_2 S_x$
			\item $S_y X_1YX_2$
			\item $S_y Y_1XY_2$
			\item $X_1YX_2 S_y$
			\item $Y_1XY_2 S_y$
		\end{itemize}
	\end{enumerate}
\end{enumerate}
